# No raw SQL strings or tagged templates

All database schema operations use drizzle-kit generated migrations. All queries use Drizzle ORM query builder. No handwritten SQL strings or sql`` tagged templates anywhere in the codebase.

Exceptions:
- `*.sql` migration files generated by drizzle-kit (not TypeScript)
- Test files that set up in-memory test databases

```grit
language js

`$tag$fragment` where {
  $fragment <: or {
    contains "CREATE ",
    contains "SELECT ",
    contains "INSERT ",
    contains "UPDATE ",
    contains "DELETE ",
    contains "ALTER ",
    contains "DROP "
  },
  $filename <: not includes ".test.",
  $filename <: not includes ".spec."
}
```
