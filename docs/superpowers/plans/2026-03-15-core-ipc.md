# core.ipc Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a typed bidirectional command bus between Bun and webview processes, replacing all `executeJavascript` / window global hacks with a clean `addMessageListener` + `send` channel.

**Architecture:** New `core.ipc` package defines typed commands and a `IpcBridge` port. `domain.adapter.rpc` implements it over Electrobun's existing IPC handle (same transport as `effect-rpc`, separate `"app-commands"` channel). `core.ui` provides a SolidJS context provider. Desktop app wires both sides.

**Tech Stack:** TypeScript, Electrobun IPC, SolidJS context

**Spec Reference:** `docs/superpowers/specs/2026-03-15-core-ipc-design.md`

**Pre-flight:** Read these files before modifying:
- `packages/libs/domain.adapter.rpc/src/model/electrobun-rpc.ts` (ElectrobunRpcHandle type)
- `packages/libs/domain.adapter.rpc/src/api/client-protocol.ts` (how addMessageListener works)
- `packages/libs/core.shared/src/rpc-schemas.ts` (MainRPCSchema)
- `packages/libs/core.ui/src/lib/runtime-provider.ts` (SolidJS context pattern)
- `packages/apps/desktop/src/bun/index.ts` (current menu handler + executeJavascript hack)
- `packages/apps/desktop/src/main-ui/index.ts` (webview init)
- `packages/apps/desktop/src/main-ui/rpc-view.ts` (webview RPC handler)

- [ ] **Step 4: Implement createElectrobunBridge**

Create `packages/libs/domain.adapter.rpc/src/api/ipc-bridge.ts`:

```typescript
import { IPC_CHANNEL, type AppCommand, type IpcBridge } from "@ctrl/core.ipc";
import type { ElectrobunRpcHandle } from "../model/electrobun-rpc";

export const createElectrobunBridge = (handle: ElectrobunRpcHandle): IpcBridge => {
  const subscribers = new Set<(command: AppCommand) => void>();

  handle.addMessageListener(IPC_CHANNEL, (raw) => {
    const command = raw as AppCommand;
    for (const handler of subscribers) {
      handler(command);
    }
  });

  return {
    send: (command) => {
      handle.send[IPC_CHANNEL](command);
    },
    subscribe: (handler) => {
      subscribers.add(handler);
      return () => {
        subscribers.delete(handler);
      };
    },
  };
};
```

import type { IpcBridge } from "@ctrl/core.ipc";
import { createContext, type ParentProps, useContext } from "solid-js";

const IpcBridgeContext = createContext<IpcBridge>();

export function IpcBridgeProvider(props: ParentProps<{ bridge: IpcBridge }>) {
  return (
    <IpcBridgeContext.Provider value={props.bridge}>
      {props.children}
    </IpcBridgeContext.Provider>
  );
}

export function useIpcBridge(): IpcBridge {
  const bridge = useContext(IpcBridgeContext);
  if (!bridge) throw new Error("IpcBridgeProvider not found");
  return bridge;
}
